# ecoMarket 🍃

ecoMarket es una tienda en línea dedicada exclusivamente a la promoción y venta de productos ecológicos, reutilizables y sustentables. El objetivo principal de este proyecto es ofrecer una interfaz accesible, limpia y atractiva que fomente el consumo responsable y la conciencia ambiental en el mercado moderno.

## 🚀 Características del Proyecto

- **Desafío de CSS3 Puro:** Interactividad avanzada, menús, estados dinámicos y validaciones de formularios resueltos al 100% sin una sola línea de JavaScript.
- **Enfoque Mobile First:** El diseño ha sido estructurado desde las pantallas más pequeñas (móviles) y adaptado progresivamente mediante Media Queries para pantallas grandes (escritorio).
- **Diseño Unificado en Dólares (USD):** Formatos de precios y catálogo alineados estrictamente con las referencias de diseño requeridas.
- **Validación Visual Nativa:** Formulario inteligente que cambia de color dinámicamente según el estado de los datos ingresados.

## Tecnologías Utilizadas

- **HTML5 Semántico:** Uso de etiquetas estructurales puras (`<header>`, `<nav>`, `<main>`, `<section>`, `<aside>`, `<article>`).
- **CSS3 Avanzado:** Implementación de variables globales, selectores avanzados, Flexbox para layouts lineales y CSS Grid para cuadrículas responsivas complejas.
- **Font Awesome 6:** Iconografía vectorial integrada por hojas de estilo.

## Estructura del Repositorio

La arquitectura del proyecto sigue estrictamente el estándar estático solicitado:

```text
EcoMercado/
├── css/
│   ├── main.css          # Reset, variables globales y tipografía
│   ├── layout.css        # Sistemas de encabezado, pie de página y cuadrículas
│   ├── components.css    # Tarjetas, botones y campos de entrada
│   └── animaciones.css   # Transiciones suaves y efectos hover
├── vistas/
│   ├── productos.html    # Catálogo con matriz de 20 productos
│   ├── sobre-nosotros.html # Historia y carrusel de testimonios
│   └── contacto.html     # Formulario inteligente y mapa estático
├── index.html            # Página de inicio oficial
└── README.md             # Documentación del proyecto
```
